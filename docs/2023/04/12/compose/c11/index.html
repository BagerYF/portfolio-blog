# How to Use Switch in Jetpack Compose

Switch is a control flow mechanism that allows you to conditionally render different views based on a value. In Jetpack Compose, you can use Switch by creating a when-else expression and providing the value to evaluate and the views to render for each case.

Here's an example of how to use Switch in Jetpack Compose:

In this example, we have a list of themes with different names and colors. We use Switch to render the currently selected theme and provide buttons to switch to the other themes. When a theme is selected, the Switch expression is evaluated, and the corresponding view is rendered.

The Switch function takes three parameters: the value to evaluate, a list of themes, and a function that is called when a theme is selected. The function takes the new theme as a parameter and updates the current theme state.

Inside the Switch expression, we use when expressions to match the value of the current theme. For each case, we render a Text view with the name and color of the selected theme. We also provide buttons to switch to the other themes. When a button is clicked, the handleThemeSelected function is called with the new theme, which updates the current theme state.

That's how you can use Switch in Jetpack Compose to conditionally render different views based on a value.
