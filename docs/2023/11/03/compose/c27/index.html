# How to Use Custom Theme in Jetpack Compose

Custom themes allow you to define and apply consistent styles throughout your Jetpack Compose UI. In this article, we'll explore how to create and use custom themes in Jetpack Compose.

# What is a Theme?

A theme in Jetpack Compose is a collection of style attributes that define the appearance of your UI elements. By default, Jetpack Compose provides a default theme that applies styles to common UI elements such as text, buttons, and cards.

# Why Use a Custom Theme?

Using a custom theme allows you to define your own styles and apply them consistently throughout your UI. This can help you create a unique and cohesive look and feel for your app.

# Common Theme Attributes

Here are some of the common theme attributes that you can customize:

  1. Color: You can customize the colors of your theme using the primary , primaryVariant , secondary , secondaryVariant , surface , background , and error properties.
  2. Typography: You can customize the font family, font size, font weight, and line height of your theme using the typography property.
  3. Shapes: You can customize the shapes of your theme using the shapes property.
  4. Content: You can customize the content of your theme using the content property.

# Summary

Custom themes allow you to define and apply consistent styles throughout your Jetpack Compose UI. In this article, we've explored how to create and use custom themes in Jetpack Compose. With custom themes, you can create a unique and cohesive look and feel for your app.
