# Common Properties of Flutter's Routing Control

Here are some common properties of Flutter's routing control that you might find useful:

  • initialRoute : This property is used to set the initial route. It takes a string value that represents the route name.
  • routes : This property is used to define the routes. It takes a map of route names to route builders.
  • navigatorKey : This property is used to set the navigator key. It takes a GlobalKey object.
