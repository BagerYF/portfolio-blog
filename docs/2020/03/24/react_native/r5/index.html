# Setting Properties

The Image component has several properties that you can set to customize its appearance and behavior. Here are some of the most commonly used properties:

  • source : Specifies the source of the image. This can be a local image file, a network URL, or a static image resource.
  • style : Specifies the style of the image. This can include properties such as width , height , resizeMode , borderRadius , etc.
  • onLoad : Specifies the function to be called when the image is loaded.
  • onError : Specifies the function to be called when an error occurs while loading the image.
