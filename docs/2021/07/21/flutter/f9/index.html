# Working with BottomNavigationBar in Flutter

1. Introduction

The BottomNavigationBar is a fundamental component in Flutter that provides a navigation bar at the bottom of the screen. It typically contains a set of icons or labels representing different app screens. In this article, we will explore how to use the BottomNavigationBar in Flutter, customize it to meet our design requirements, and utilize common properties.

3. Customizing the BottomNavigationBar

The BottomNavigationBar can be customized to match the design of your application. Here are some common customization options:

  • Icon and Label: You can set the icon and label of each BottomNavigationBarItem using the icon and label properties. It can be a widget or any other widget you want to display as the icon.
  • Selected Icon and Label: You can customize the icon and label of the selected item using the selectedIcon and selectedLabel properties.
  • Background Color: You can set the background color of the BottomNavigationBar using the backgroundColor property. This determines the color of the bar behind the icons and labels.
  • Active Color: You can customize the color of the selected item using the activeColor property. This determines the color of the icon and label when the item is selected.
  • Item 间距: You can customize the spacing between items using the itemSpacing property. This determines the horizontal spacing between each item.
  • Icon Size: You can customize the size of the icons using the iconSize property. This determines the size of the icons displayed in the BottomNavigationBar.

In the above code, we customized the BottomNavigationBar by setting the background color to blue, the active color to white, the item spacing to 8, and the icon size to 32. We also changed the icon and label of the selected item.

5. Common Properties of the BottomNavigationBar

Here are some common properties of the BottomNavigationBar that you may find useful:

  • currentIndex : Specifies the index of the currently selected item. This property can be used to programmatically change the selected item.
  • onTap : Specifies the callback function that is called when an item is tapped. This function receives the index of the tapped item.
  • automaticallyImplyLeading : Specifies whether the leading item should be automatically implied based on the platform's design guidelines.

In the above code, we customized the BottomNavigationBar by setting the currentIndex property to control the selected item programmatically. We also added an onTap callback function to handle the item taps and update the currentIndex value.

6. Conclusion

In this article, we explored how to use the BottomNavigationBar in Flutter. We learned how to create a basic BottomNavigationBar, customize it to match our design requirements, and use it as a layout widget. By understanding and utilizing the common properties of the BottomNavigationBar, you can create beautiful and functional navigation bars for your mobile applications.
